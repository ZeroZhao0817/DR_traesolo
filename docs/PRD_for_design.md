## 1. 产品概述

### 1.1 产品定位
一款轻量级桌面番茄钟提醒工具，帮助长时间电脑工作者定时休息。应用通过主窗口承载计时与设置，通过独立提醒窗口展示放松动作、名言和阶段确认。

### 1.2 目标用户
- 程序员、开发者等长时间面对电脑工作的人群
- 需要简单、低干扰番茄钟提醒的桌面用户

### 1.3 设计原则
- **极简设计**：固定小窗口，少干扰，启动快
- **明确反馈**：工作结束后必须确认休息或输入确认语继续工作
- **双主题**：支持浅色和深色主题，跟随系统
- **清晰的色彩区分**：工作和休息阶段有明显的色彩区别

## 3. 界面设计规范

### 3.1 主窗口

#### 窗口属性
| 属性 | 值 |
|-----|----|
| 窗口尺寸 | 360×500px |
| 可缩放 | 否 |
| 标题栏 | 隐藏，自定义标题栏 |
| 窗口置顶 | 可选 |

#### 自定义标题栏
- **左侧**：22×22px 圆形图标（主色填充）+ 应用名称“久坐提醒 v1.1.0”
- **右侧**：四个操作按钮（从左到右）：
  1. 置顶按钮（图标：`push_pin` / `push_pin_outlined`）
  2. 悬浮球按钮（图标：`picture_in_picture_alt_outlined`）
  3. 设置按钮（图标：`settings_outlined`）
  4. 关闭按钮（红色背景，图标：`close`）
- 按钮尺寸：26×26px，圆角 5px
- 背景色：浅色主题为白色，深色主题为 `#202043`
- 拖动区域：整个标题栏可拖动

#### 主界面布局
1. **状态标签**：显示当前阶段（工作中/休息中），带彩色背景圆点
2. **圆形进度环**：
   - 尺寸：158×158px
   - 线宽：7px
   - 进度显示：环形进度条 + 中心倒计时数字
   - 中心区域：倒计时数字 + 小图标 + 阶段文字
3. **控制按钮**：
   - 重置按钮（左，62×30px）
   - 主按钮（中，90×36px）- 开始/暂停
   - 跳过按钮（右，62×30px）
4. **模式选择**：
   - 快速模式：25+5 / 45+10 / 60+15
   - 自定义按钮
5. **底部文字**：“------ 专注工作，按时休息 ------”

### 3.2 悬浮球模式

#### 窗口属性
| 属性 | 值 |
|-----|----|
| 窗口尺寸 | 150×150px |
| 背景 | 透明 |
| 可缩放 | 否 |

### 3.3 独立提醒窗口

#### 窗口属性
| 属性 | 值 |
|-----|----|
| 窗口尺寸 | 屏幕 80%（宽高） |
| 位置 | 居中 |
| 窗口置顶 | 是 |
| 标题栏 | 隐藏，自定义标题栏 |

#### 布局结构（左右分栏）
**左侧 - 放松动作区**（占主要空间）
- 标题栏：“放松动作” + 四图/单图切换按钮
- 动作图片展示区
  - 四图模式：2×2 网格
  - 单图模式：单张大图 + 左右导航按钮
- 图片资源：
  - 浅色主题：`lighti_actions_01.png` 到 `lighti_actions_04.png`
  - 深色主题：`dark_actions_01.png` 到 `dark_actions_04.png`

**右侧 - 信息与操作区**（右侧）
- **每日激励**卡片
  - 标题栏：“每日激励” + “毛选摘录”标签 + 刷新按钮
  - 名人名言展示：带引号，左对齐
  - 作者：右对齐
- **操作区**
  - 提示文字
  - （工作结束时）确认语文本输入框
  - 两个操作按钮（50px 高）：
    - 次要按钮（继续工作/继续休息）
    - 主要按钮（开始休息/开始工作）

#### 休息进行中状态
当用户选择“开始休息”后，右侧区域显示进度面板：
- 休息进度条
- 剩余时间
- 完成百分比

  2. 桌面通知卡片
     - 图标（notifications_outlined，绿色）
     - 开关按钮
- **底部提示**：“设置将自动保存到本地”

#### 自定义番茄钟弹窗
- 两个步进器：工作时长、休息时长
- 范围提示文字
- 取消/确定按钮

### 3.5 休息结束确认卡片

#### 视觉设计
- **背景**：毛玻璃效果（BackdropFilter）
- **卡片**：圆角矩形，居中
- **内容**：
  - 图标（coffee）+ 标题“休息结束” + 轮次信息
  - 休息结束提示文字
  - 倒计时显示（15秒自动开始）
  - 两个按钮：继续休息 / 开始工作

## 4. 交互设计规范

### 4.1 阶段切换逻辑
```
工作计时 → 工作结束 → 提醒窗口弹出
                        ↓
               [开始休息]  ←──────┐
                        ↓         │
               休息计时          输入确认语
                        ↓         │
               休息结束 →  [继续工作]
                        ↓
               下一轮工作
```

### 4.2 按钮交互状态

#### 主按钮（开始/暂停）
- 状态：开始（工作阶段紫色，休息阶段绿色）/ 暂停
- 高度：36px（主窗口），50px（提醒窗口）

#### 次要按钮
- 边框按钮
- 高度：38px（弹窗），50px（提醒窗口）

#### 小按钮
- 高度：30px

### 4.3 确认语机制
工作结束后，若用户选择“继续工作”，需要输入固定确认语：
- **确认语文本**：健康不要了，继续工作
- **按钮状态**：输入正确前禁用，输入正确后启用

### 4.4 悬浮球切换
- 点击进度环区域可切换到悬浮球
- 点击悬浮球可切换回主窗口

## 5. 视觉设计细节

### 5.1 进度环设计
- **环形进度条**：使用 CircularProgressIndicator
- **线端**：StrokeCap.round（主窗口），StrokeCap.butt（悬浮球）
- **背景环**：半透明主色
- **光晕**：主色半透明阴影

### 5.2 卡片设计
- 圆角：12-16px
- 边框：1px 边框线
- 阴影：根据主题调整

### 5.3 图标风格
- 使用 Material Design Icons
- 图标大小：12-22px
- 颜色：与当前阶段主题色一致或灰色
